# 📝 Quick Start (5 Phút)

## Giả sử bạn đã cài Node.js & MySQL

### 1️⃣ Backend Setup (Terminal 1)
```bash
cd backend
npm install
cp .env.example .env
# ⚠️ Sửa .env với MySQL password của bạn
npm run setup
npm run dev
```

✅ Sẽ thấy: `🚀 Server is running on port 5000`

### 2️⃣ Frontend Setup (Terminal 2)
```bash
npm install
npm run dev
```

✅ Sẽ thấy: `http://localhost:5173`

### 3️⃣ Mở Browser
http://localhost:5173 → Đăng ký & Đăng nhập

**Done! 🎉**
